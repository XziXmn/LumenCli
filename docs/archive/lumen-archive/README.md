# Legacy Lumen Archive

这个目录来自仓库根部原先的 `.lumen-archive/`。

它保存的是 Lumen 早期 fork 阶段留下的历史材料，包括：

- 早期设计 / 计划 / 报告草稿
- 已废弃的原型代码片段
- 旧的 `novel-example` 示例目录
- 初期 fork 时保留的原始 README 副本

用途：

- 仅供历史追溯和人工检索
- 不作为当前实现、当前文档体系或 merge 决策的真源
